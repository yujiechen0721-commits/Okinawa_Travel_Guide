# 沖繩旅遊指南 Okinawa Travel Guide

一個用純HTML製作的沖繩旅遊資訊網站，  
涵蓋景點、美食、住宿、交通、節慶、本地文化等完整攻略，適合自由行旅客快速查詢！

## 網站預覽
線上立即瀏覽  
https://yujiechen0721-commits.github.io/Okinawa_Travel_Guide/

## 主要內容
- 沖繩簡介與地理位置
- 景點推薦(主題公園、海灘、名勝古蹟、博物館…)
- 美食＆當地小吃＆特色餐廳
- 住宿推薦(飯店、民宿、度假村、青年旅社)
- 交通方式、旅遊安全、購物、本地習俗
- 活動與節日一覽
